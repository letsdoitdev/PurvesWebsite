# Purves for Congress 2026

Campaign website for **Arthur Purves**, candidate for U.S. Congress in
Virginia's 11th Congressional District (2026).

## The Message

A bipartisan **Make America Healthy Again (MAHA)** platform: **Cut the Deficit
Without Cutting a Program** by ending the chronic disease epidemic that is
bankrupting the country. Not left. Not right. Just healthy.

## The Site

A single, self-contained `index.html` file with embedded CSS and a small amount
of vanilla JavaScript (mobile menu toggle only). No build step and no
frameworks required.

- **Fonts:** Google Fonts (Inter + Montserrat)
- **Icons:** Font Awesome 6 (CDN)
- **Layout:** CSS Grid + Flexbox, fully mobile responsive
- **Features:** sticky header, smooth-scroll navigation, big bold stat cards

### Viewing locally

Open `index.html` in any modern browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
